## ðŸ“‚ Repository Structure

### Data & Lists
* **`sparc_raw_full_3389_points.csv`**: The complete raw rotation curve dataset from the SPARC database.
* **`sparc_subset_118_list.txt`**: The list of high-quality Galaxy IDs used for the analysis.
    * *Note:* While the filename references the original 118-galaxy sample, the list actually contains **125 unique high-quality galaxies**. The analysis code processes all 125 galaxies to ensure maximum statistical power.

### Code & Analysis
* **`analyze.py`**: The main scientific script. It reads the raw data, applies physical filters (Radius > 0), and performs Dynamic Nested Sampling (Dynesty) to constrain the leakage index ($n$) and scale ($a_5$).
* **`chain_results.txt`**: The raw posterior chain data generated by the code.
* **`analysis_summary.txt`**: A summary of the statistical results (Mean parameters, Log-Evidence).
* **`class_mu_patch.diff`**: A diff file to patch the Boltzmann code **CLASS** for non-local gravity modifications.

### Visualizations
* **`Figure_1` - `Figure_5`**: Official figures illustrating rotation curve fits, corner plots, and cosmological constraints.
